# Поиск документов

## Как быстро находить нужные файлы

Одна из главных сильных сторон DocManager — мощный и быстрый поиск.

### Виды поиска

- **По имени файла** — стандартный поиск по названию.
- **По тегам** — выберите один или несколько тегов в боковой панели.
- **Полнотекстовый поиск** — ищет текст внутри содержимого документов (особенно полезно для PDF).

![Экран поиска с результатами](/assets/search-screen.png)

## Как пользоваться поиском

1. Нажмите на строку поиска в верхней части главного экрана.
2. Введите запрос (например, «договор Иванов» или «счёт 2025»).
3. Результаты появляются мгновенно.
4. Нажмите на найденный файл для просмотра.

### Полезные приёмы поиска

- Используйте кавычки для точной фразы: `"акт выполненных работ"`.
- Комбинируйте с тегами: сначала выберите тег `Клиент_Петров`, потом ищите внутри.
- Ищите по номеру счёта, дате, сумме — всё, что есть в тексте документа.

## Примеры реальных ситуаций

- Вы на встрече с клиентом и нужно быстро показать договор → ищите по имени клиента или номеру договора.
- Налоговая запросила отчёт за квартал → ищите по тегу `Налоговая` и году.
- Нужно найти чек на покупку оборудования → ищите слово «чек» или название магазина.

### Советы по оптимизации поиска

- Присваивайте осмысленные имена файлам при сохранении.
- Добавляйте описание в свойствах файла (в формате Markdown) — оно тоже индексируется.
- Регулярно очищайте архивные документы в отдельную папку.
