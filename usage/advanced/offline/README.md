# Оффлайн-режим и синхронизация

DocManager работает полностью оффлайн, но поддерживает синхронизацию с сервером.

## Как работает оффлайн

- Все файлы хранятся локально.
- Поиск и просмотр без интернета.

### Синхронизация

- Подключите сервер в настройках.
- Нажмите "Синхронизировать".

- **Шаги:**
  - Выберите коллекцию.
  - Подтвердите изменения.
