# DocManager — Система электронного документооборота для индивидуального предпринимателя

Добро пожаловать в официальную пользовательскую документацию мобильного приложения **DocManager**!  

Это приложение создано специально для индивидуальных предпринимателей, чтобы упростить работу с документами: договорами, счетами, актами выполненных работ, чеками, перепиской и другими файлами. Вы можете работать полностью оффлайн, быстро находить нужные документы и удобно организовывать их с помощью тегов.

![Главный экран приложения](/assets/main-screen.png)

## Основные преимущества приложения

- Полностью оффлайн — все документы хранятся локально на вашем устройстве.
- Гибкая классификация с помощью цветных тегов (один документ — множество тегов).
- Полнотекстовый поиск по содержимому файлов (даже внутри PDF).
- Встроенные просмотрщики для PDF, изображений, текстовых файлов.
- Поддержка синхронизации с собственным сервером (опционально).
- Не требует регистрации, подписки или интернета для основной работы.

## Быстрый старт

1. Установите приложение из APK-файла.
2. При первом запуске выберите папку на устройстве, где хранятся ваши документы.
3. Приложение автоматически просканирует папку и создаст миниатюры.
4. Начните добавлять теги и использовать поиск!

### Полезные советы для начала работы

- Рекомендуем создать отдельную папку только для бизнес-документов (например, `/Documents/Business/`).
- Используйте теги вроде `2025`, `Оплачено`, `Договор`, `Клиент_Иванов` для быстрой фильтрации.
- Включите тёмную тему в настройках для комфортной работы вечером.
