# О приложении DocManager

## Что это за приложение и для кого оно

**DocManager** — мобильное приложение для Android, разработанное специально для индивидуальных предпринимателей (ИП). Оно помогает организовать электронный документооборот без использования дорогих облачных сервисов и корпоративных систем.

По данным на 2025 год в России более 4,5 миллионов активных ИП. Большинство из них ежедневно работают с большим количеством документов: договоры, счета-фактуры, акты, чеки, переписка с клиентами и поставщиками. Традиционное хранение в папках или облаке часто приводит к потере времени на поиск и риску потери важных файлов.

DocManager решает эти проблемы, предлагая полностью локальное, бесплатное и удобное решение.

![Диаграмма прецедентов из курсового проекта](../assets/use-case-diagram.png)

## Основные возможности

- **Локальное хранение** — все файлы и метаданные остаются на вашем устройстве.
- **Множественные теги** — один документ может относиться к нескольким категориям одновременно.
- **Полнотекстовый поиск** — ищет не только по имени, но и по тексту внутри PDF и офисных файлов.
- **Встроенные просмотрщики** — открывайте PDF, изображения, текст без дополнительных приложений.
- **Оффлайн-режим по умолчанию** — приложение работает в любом месте: в дороге, в самолёте, в зоне без интернета.
- **Опциональная синхронизация** — с собственным сервером для резервного копирования.

## Почему это удобно именно для ИП

- Нет ежемесячной платы (в отличие от Диадок, СБИС, 1С-Документооборот).
- Полная конфиденциальность — данные не уходят на чужие серверы.
- Мобильность — быстрый доступ к документам во время встреч с клиентами.
- Простота — не нужно быть IT-специалистом для начала работы.
