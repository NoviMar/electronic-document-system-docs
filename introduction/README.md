# О приложении DocManager

DocManager — мобильное приложение для индивидуальных предпринимателей, разработанное для эффективного управления документами. Оно решает проблемы с хранением, поиском и организацией файлов в бизнесе.

## Цель и задачи

Приложение помогает ИП:

- Организовывать документы без интернета.
- Искать файлы по тексту внутри них.
- Классифицировать с помощью тегов.

### Предметная область

Индивидуальные предприниматели (более 4,5 млн в РФ на 2025 год) работают с договорами, счетами, актами. DocManager упрощает это.

- **Преимущества:**
  - Оффлайн-доступ.
  - Гибкие теги.
  - Просмотр PDF, изображений и т.д.
